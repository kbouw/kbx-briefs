### 2. Anti-patterns: things to avoid

Don't submit unreviewed AI-generated code for pull requests—you're just delegating the actual work to reviewers who could have prompted the agent themselves. Include evidence of manual testing, keep changes small, and ensure you've validated both the code functionality and any AI-written PR descriptions before asking others to spend time reviewing.
