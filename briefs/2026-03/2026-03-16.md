### 1. What is agentic engineering?

Simon Willison defines 'agentic engineering' as developing software with coding agents that can both write and execute code in loops until goals are met, distinguishing it from Karpathy's 'vibe coding' by emphasizing code execution as the defining capability that enables iterative improvement toward demonstrably working solutions.
