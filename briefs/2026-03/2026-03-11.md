## AI/ML

### 1. AI should help us produce better code

Coding agents excel at simple but time-consuming refactoring tasks that typically create technical debt—API changes across dozens of files, nomenclature cleanup, and code splitting—dropping the cost so low that teams can maintain zero tolerance for code smells. The key is adopting Every's "compound engineering" approach: retrospectives after each agent project to document what worked, continuously improving both codebase quality and agent instructions.
