### 1. 🔴 Datasette Agent

Simon Willison released Datasette Agent, the first AI assistant that can conversationally query SQLite databases and generate charts through an extensible plugin system. This bridges his 3-year LLM library work with Datasette, enabling natural language database queries that compile to SQL and opening up agent-based data analysis workflows.
