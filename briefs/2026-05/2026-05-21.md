### 3. The famous o3 "GeoGuessr" prompt did not work

Author benchmarked Kelsey Piper's famous o3 GeoGuessr prompt against a basic prompt on 200 images and found the elaborate prompt actually performed worse (102.3km vs 83.2km median distance). This demonstrates how easy it is to fool yourself about prompt engineering effectiveness when models are already capable—rigorous benchmarking beats vibes.

**Source:** [Sean Goedecke](https://seangoedecke.com/the-o3-geoguessr-prompt-did-not-work/)
**Tags:** o3, prompt engineering, benchmarking, geolocation

### 4. How fast is 10 tokens per second really?

Mike Veerman built an HTML simulator that shows what different LLM token speeds (5-800 tokens/second) actually look like in practice, helping developers understand the real-world implications of advertised model speeds like "30 tokens/second."

**Source:** [Simon Willison's Weblog](https://simonwillison.net/2026/May/20/tokens-per-second/#atom-everything)
**Tags:** LLM performance, token speed, benchmarking
