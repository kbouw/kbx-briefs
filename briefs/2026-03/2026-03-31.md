### 1. Quoting Georgi Gerganov

Georgi Gerganov explains that local coding agents fail not due to model capability issues, but because of fragile infrastructure chains—buggy inference engines, broken chat templates, and poor prompt construction across components built by different teams create subtle failures that users don't realize are breaking their workflows.

### 2. Ollama is now powered by MLX on Apple Silicon in preview

Ollama now supports Apple's MLX framework for running local LLMs on Apple Silicon, potentially offering better performance and efficiency compared to the previous CPU/GPU execution paths on Mac hardware.

**Source:** [Hacker News Best](https://ollama.com/blog/mlx)
**Tags:** Ollama, MLX, Apple Silicon, local LLM

### 3. Mr. Chatterbox is a (weak) Victorian-era ethically trained model you can run on your own computer

Trip Venturella released Mr. Chatterbox, a 340M parameter LLM trained solely on 28,000 Victorian-era books from the British Library, proving copyright-free model training is possible but revealing it needs ~4x more data to be conversationally useful. The author used Claude Code to build a complete LLM plugin from scratch, demonstrating AI-assisted development for local model deployment.

**Source:** [Simon Willison's Weblog](https://simonwillison.net/2026/Mar/30/mr-chatterbox/#atom-everything)
**Tags:** Victorian-era training data, local LLMs, copyright-free models, LLM plugin development
