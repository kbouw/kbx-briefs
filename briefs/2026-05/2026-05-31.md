# Daily Intelligence Brief — May 31, 2026

## AI/ML

### 1. Build agents, not pipelines

Author argues agents outperform pipelines for complex LLM tasks because they can dynamically gather context, while RAG-based pipelines fail since "finding relevant information is often as hard as solving the problem itself." The unpredictability of agent costs is worth accepting for tasks like coding where pipeline approaches simply cannot work.

**Source:** [Sean Goedecke](https://seangoedecke.com/build-agents-not-pipelines/)
**Tags:** LLM agents, RAG, pipeline architecture, context gathering
