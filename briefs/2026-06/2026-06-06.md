### 2. 🔴 OpenAI Help: Lockdown Mode

OpenAI's new Lockdown Mode blocks outbound network requests to prevent data exfiltration in prompt injection attacks, targeting the easiest leg of the "Lethal Trifecta" (private data access + untrusted content + exfiltration vectors). This implicitly confirms that default ChatGPT settings don't robustly protect against determined data theft attacks.
