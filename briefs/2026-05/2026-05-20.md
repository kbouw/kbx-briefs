### 2. Prompts are technical debt too

Prompts are worse technical debt than code because they decay silently with every model upgrade—carefully crafted prompts from January may be harmful by February. Engineers should use third-party AI tools like Cursor or Copilot with minimal configuration rather than building bespoke prompt setups, since dedicated teams can better handle the constant re-optimization needed for each model release.

**Source:** [Sean Goedecke](https://seangoedecke.com/prompts-are-technical-debt-too/)
