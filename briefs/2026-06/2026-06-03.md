## AI/ML

### 1. 🔴 Microsoft's new MAI models

Microsoft released MAI-Thinking-1 (1T parameters, 35B active) claiming to outperform Sonnet 4.6 and MAI-Code-1-Flash (137B, 5B active) for GitHub Copilot, but despite claims of "clean, licensed data," both models actually trained on massive web crawls including Common Crawl with the same licensing issues as other major LLMs.

### 2. 🔴 MAI-Code-1-Flash

Microsoft launched MAI-Code-1-Flash as part of seven new MAI models, positioning it as a specialized code generation model. The release signals Microsoft's push into competitive coding AI space against established players like GitHub Copilot and Claude.

**Source:** [Hacker News Best](https://microsoft.ai/news/introducingmai-code-1-flash/)
**Tags:** MAI-Code-1-Flash, Microsoft AI, code generation, model release

### 3. Uber Caps Usage of AI Tools Like Claude Code to Manage Costs

Uber caps employee AI coding tool usage at $1,500/month per tool after burning through their 2026 AI budget in four months, setting limits equivalent to ~11% of median engineer compensation ($36k annual cap vs $330k salary). This suggests enterprise token costs are 10x higher than individual subscriber rates, making $1,500/month caps more sustainable than unlimited usage policies that encouraged token-burning competitions.
